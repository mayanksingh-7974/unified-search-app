# KnowFlow AI 🧠

**AI-Powered Unified Knowledge Assistant**

> Search across Slack, Notion, and Google Drive instantly. Get answers in seconds, not hours.

---

## 🎯 Problem & Solution

**Problem**: Employees waste 2+ hours daily searching across 5+ disconnected platforms (Slack, Notion, Google Drive, SharePoint) for scattered information.

**Solution**: One unified search powered by AI that finds answers across all platforms instantly with verified sources.

---

## ✨ Key Features

- 🔍 **Unified Search** - One query searches Slack, Notion, and Google Drive
- 💬 **Natural Language** - Ask questions like you'd chat with a colleague
- 📎 **Source Citations** - Every answer includes verified sources
- ⚡ **Lightning Fast** - Get answers in seconds
- 📱 **Mobile Ready** - Works on desktop, tablet, and mobile

---

## 🏗️ Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS, Lucide React
- **Backend**: Node.js, Next.js API Routes
- **Deployment**: Vercel
- **Future**: Slack API, Notion API, Google Drive API, OpenAI

---

## 🚀 Quick Start

```bash
# Clone and setup
git clone https://github.com/mayanksingh-7974/unified-search-app
cd unified-search-app

# Install dependencies
npm install

# Start development server
npm run dev
```

Visit **http://localhost:3000** and click "Try Now"

**Test queries:**
- "Tell me about API documentation"
- "What are the Q4 goals"
- "Financial report"
- "Project timeline"
